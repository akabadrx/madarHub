# Madar Hub Website

Static responsive website for `madarorbit.com`.

## Pages

- `index.html`
- `about.html`
- `pricing.html`
- `events.html`
- `badr-academy.html`
- `contact.html`

## Assets

- Shared CSS: `assets/styles.css`
- Shared JavaScript: `assets/script.js`
- Local generated visuals: `assets/madar-orbit-visual.png`, `assets/founder-placeholder.png`, `assets/kimironko-map-placeholder.png`

Open `index.html` directly in a browser, or serve the folder with any static web server.

## Deploy

Double-click `deploy.bat` from Windows to package the site and upload it to
`root@72.62.132.37:/var/www/madarorbit`.

For a local packaging test without uploading, run:

```bat
deploy.bat --dry-run
```
